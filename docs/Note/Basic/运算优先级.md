# 运算优先级

运算优先级是只当遇到两个运算符，该先运算哪个。当两个运算符是同级别时，我们将从左到右运算，否则优先级高的先运算。

下面我将列出一个表格，说明运算符的优先级：

| 优先级  | 运算符                   | 描述                 | 运算方向    |
| ---- | --------------------- | ------------------ | ------- |
| 1 级  | （），a++，a--            | 括号，后缀自增/自减         | $\to$   |
| 2 级  | ++a，--a，-a，！，~        | 前缀自增/自减，负号，逻辑非，位取反 | $\gets$ |
| 3 级  | $\times$， $\div$，$\%$ | 乘，除，取模             | $\to$   |
| 4 级  | $+$ ，$-$              | 加，减                | $\to$   |
| 5 级  | <<，>>                 | 左移，右移              | $\to$   |
| 6 级  | \==，!=                | 相等比较               | $\to$   |
| 7 级  | &                     | 按位与                | $\to$   |
| 8 级  | ^                     | 按位异或               | $\to$   |
| 9 级  | \|                    | 按位或                | $\to$   |
| 10 级 | &&                    | 逻辑与                | $\to$   |
| 11 级 | \|\|                  | 逻辑或                | $\to$   |
| 12 级 | ？：                    | 三元运算符              | $\gets$ |
| 13 级 | =                     | 赋值                 | $\gets$ |
|      | +=，-=，\*=，/=          | 复合赋值               | $\gets$ |
|      | %=，<<=，>>=，&=         | 复合赋值               | $\gets$ |
|      | ^=，\|=                | 复合赋值               | $\gets$ |
| 14 级 | ，                     | 逗号                 | $\to$   |

**备注：上图表格级别越小，优先级越高**
